## Run Locally

Clone the project

```bash
  git clone https://github.com/ArmanKalmakhambetov/editor_front_kazniisa.git
```

Go to the project directory

```bash
  cd editor_front_kazniisa
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
